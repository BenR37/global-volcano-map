About this tool

The Global Volcano Map is an interactive open-source dashboard for exploring and filtering the world's volcanoes. It visualises over 2,600 volcanoes from the Holocene and Pleistocene epochs, with data on eruption history, elevation, tectonic setting, rock type, and more.

Use the filter panel to narrow by continent, country, geological period, volcano type, or eruption year range. The analytics panel charts eruption trends and type breakdowns for whatever subset you've filtered. Any view can be shared as a permalink or exported as a CSV.

Data source

Volcano data is sourced from the Smithsonian Institution Global Volcanism Program (GVP) — the authoritative global database of Holocene and Pleistocene volcanism. This tool is an independent project and is not affiliated with or endorsed by the Smithsonian Institution.
